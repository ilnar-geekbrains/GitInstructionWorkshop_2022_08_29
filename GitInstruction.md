### Базовая работа с ветками в git.
1. Чтобы посмотреть какие ветки существуют - git branch

2. Чтобы создать новую ветку - git branch name_branch

3. Чтобы перейти на другую ветку - git checkout name_branch

4. Слить ветку в текущую ветку - git merge name_branch

5. Чтобы удалить ветку - git branch -d name_branch
